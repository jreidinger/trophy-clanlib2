4.2  Speed Definition

I assume that you have already finished drawing the visual map of your track on layer vmap. Activate now the layer "fmap".

Set the active color to R 0 G 240 B 0 (#00f000) and fill all parts where cars can drive with full speed with this color. Make sure that you use tools which create exact contours and not smooth fades.

Set the active color to R 0 G 0 B 0 (#000000) and fill all parts where cars cannot drive at all.

You might also want to define areas where cars can drive but not with full speed. You can color those areas with the colors between #000000 and #00f000. E.g. #008000 means that cars can drive there with 50% of their maximum speed. You can use this for parts beside the road to prevent drivers from taking a shortcut. See also chapter "Track Path Definition" for learning how to define the track path players must take to fulfill a lap.
